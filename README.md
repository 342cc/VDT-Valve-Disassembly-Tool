# VDT – Valve Disassembly Tool

**Open Source Hardware – CC BY-SA 4.0**

## Description
Universal, bidirectional mechanical tool for engine valve disassembly and reassembly. Designed for low-resource environments.

- Electromagnetic collet capture
- Universal adjustments (X, Y, Z, tilt)
- Web assistant (PWA) for guidance

## Reproducibility
All CAD files, Bill of Materials (BOM), and web app code are provided to enable local manufacturing.
